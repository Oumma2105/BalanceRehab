\# MVP Scope



\## Build First



1\. Dashboard

2\. Patient management

3\. Patient profile

4\. New test workflow

5\. Live assessment page with demo data

6\. Results page

7\. PDF report generation

8\. Progress tracking

9\. EN/FR language toggle



\## Test Types



\### Static Test

Support ring installed.



\### Dynamic Test

Support ring removed.



\## Demo Mode First



The first version must work without:

\- ESP32

\- ultrasonic sensors

\- webcam



Use mock data first.



\## Do Not Build Yet



\- login/authentication

\- Raspberry Pi deployment

\- Wi-Fi ESP32 communication

\- advanced machine learning

\- exact session prediction

\- cloud database

\- mobile app



\## Priority



A polished working app is more important than advanced features.
